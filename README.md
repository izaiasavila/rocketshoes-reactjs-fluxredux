Projeto em ReactJS para testar a arquitetura Flux (redux).

## Inciar o projeto

# Instalar dependências:

`yarn install`

# Iniciar o servidor teste

`json-server server.json -p 3333`

# iniciar a aplicação ReactJS

`yarn start`
